# 🚦 Urban Flow

Urban Flow is a smart traffic simulation platform designed to optimize traffic flow using intelligent signal control and a futuristic UI.

## 🔥 Features
- Real-time traffic simulation
- AI-style assistant dashboard
- Dynamic signal control
- Modern UI with particle effects
- Emergency override system

- JavaScript
